# 贡献者行为准则

## 我们的承诺

为了让每个人都能在开放、友善的环境中参与本项目，我们承诺：

- 使用包容、友好的语言
- 尊重不同的经验和观点
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表现出同理心

## 不可接受的行为

- 使用性暗示或不当的语言或图像
- 人身攻击、侮辱或贬损的评论
- 公开或私下骚扰
- 未经明确许可发布他人隐私信息
- 其他在专业环境中不被认为是合适的行为

## 执行责任

项目维护者有权利和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、Issues 等，并可对违规者实施临时或永久封禁。

## 适用范围

本准则适用于项目空间及在公共场合代表项目时的个人。

## 联系方式

若发现违规行为，可通过 [GitHub Issues](https://github.com/gloweaseco-leo/clawme-skill-maker/issues) 私密联系维护者。
